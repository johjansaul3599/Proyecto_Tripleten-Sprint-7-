# Análisis de Datos de Vehículos - Sprint 7

Este proyecto consiste en una **aplicación web interactiva** desarrollada con **Python** y **Streamlit**. El objetivo principal es proporcionar una herramienta visual para explorar un conjunto de datos sobre anuncios de venta de vehículos en los EE. UU.

## Propósito del Proyecto
La aplicación sirve para realizar un Análisis Exploratorio de Datos (EDA) de manera rápida y sencilla, permitiendo a los usuarios identificar patrones o tendencias en los precios y el kilometraje de los vehículos sin necesidad de escribir código.

## Funcionalidades
La aplicación web ofrece las siguientes herramientas interactivas:
* **Visualización de Distribución**: Mediante un botón, el usuario puede generar un **histograma** dinámico para analizar la frecuencia de los valores en la columna de odómetro (kilometraje).
* **Análisis de Relaciones**: Incluye un botón para construir un **gráfico de dispersión** (scatter plot) que permite visualizar la relación entre el kilometraje y el precio de los vehículos.
* **Interactividad Completa**: Gracias a Plotly, los gráficos permiten hacer zoom, filtrar categorías y obtener detalles al pasar el cursor sobre los datos.

## Tecnologías Utilizadas
* **Python 3.11**
* **Pandas**: Para la manipulación y carga de datos.
* **Streamlit**: Para la creación de la interfaz web.
* **Plotly Express**: Para la generación de gráficos interactivos.
